Setup the project:

```bash
git clone https://github.com/Pokapiec/cinema-backend.git
```

```bash
python -m venv .venv
```

```bash
.venv/Scripts/activate  # Windows
```

```bash
source .venv/bin/activate  # Linux
```

```bash
pip install -r requirements.txt
```

Developement server runs as usual:

```bash
uvicorn main:app --reload
```
